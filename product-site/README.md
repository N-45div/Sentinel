# SentinelX Product Site (Next.js)

- Next.js + TailwindCSS
- Hero: Build autonomous, verifiable agents on Solana
- Tech marquee: Visa (TAP), x402, XMCP (removed unused sponsors)
- Features: x402 Gateway, Autonomous & Verifiable (TAP/Visa), Wallets & SPL

## Run locally

```
cd product-site
npm i
npm run dev
```

Site runs on http://localhost:4000

## Build

```
npm run build
npm start
```
